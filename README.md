# e2e-testing-boilerplate
An E2E automation boilerplate using Playwright and TypeScript. Features Page Object Model (POM), CI/CD integration, and API-driven testing.

This repository serves as a E2E testing boilerplate designed for scalability, maintainability, and speed. Built with Playwright and TypeScript, it demonstrates modern automation best practices including

Architecture: Page Object Model (POM) for clean, reusable code.

CI/CD: Integrated GitHub Actions workflow for automated test execution on every PR.

Testing Strategy: Hybrid approach combining UI testing with API data seeding.

Reporting: Detailed HTML reports with traces, screenshots, and video on failure.

Reliability: Implements smart locators and auto-awaiting to eliminate flakiness.
